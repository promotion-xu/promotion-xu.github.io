## Commit

| emoji       | code          | commit 说明    |
| ----------- | ------------- | -------------- |
| 🎉(庆祝)    | `:tada:`      | 初次提交       |
| 📝 (备忘录) | `:memo:`      | 撰写文档       |
| 🚀 (火箭)   | `:rocket:`    | 部署功能       |
| ✨ (火花)   | `:sparkles:`  | 引入新功能     |
| 🚑 (急救车) | `:ambulance:` | 重要补丁       |
| 🐛 (bug)    | `:bug:`       | bug            |
| 🔥 (火焰)   | `:fire:`      | 移除代码或文件 |
| ⚡ (闪电)   | `:zap:`       | 提升性能       |

examples:

- :rocket: feat: completed xxx feature
- :bug: fix: xxx bug
- :zap: perf: xxx performance
- :memo: docs: xxx document
